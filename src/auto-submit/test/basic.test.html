auto-submit: basic (no DOM morphing)

Only Last has data-pwc-auto-submit and triggers a submission on change. First and Color are plain fields that are only submitted as part of the form data.

Without DOM morphing, the server response replaces the entire fragment. This means any unsaved local edits in plain fields are overwritten — unless they were already reflected in the submitted form data (i.e. changed before the trigger fired).
