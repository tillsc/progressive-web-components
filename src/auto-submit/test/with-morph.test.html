auto-submit: with-morph

Only Last has data-pwc-auto-submit and triggers a submission on change. With DOM morphing active, the fragment is patched in-place rather than replaced — local edits in plain fields and the current focus are preserved across the round-trip.

Two special field types are also demonstrated: Readonly computed is calculated server-side and always updated. Forced carries data-pwc-force-value, so the server value always wins even though the field is editable.
